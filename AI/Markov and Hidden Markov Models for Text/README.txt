To run this script, you will need to keep all the text files for the unigram,bigram, and trigram counts in the same directory as the Markov_Model.py script. You can run it either by opening it in an IDE and running the file or by using a shell to run it. First cd to the script's directory and run it using:

python ./Markov_Model.py


You will need the following imports: 

numpy
Levenshtein

The report holds a reference to where the Levenshtein import was taken from.

Sentence Generation should be nearly instantaneous, but sentence correction may take up to 5 minutes per sentence
to be corrected.
